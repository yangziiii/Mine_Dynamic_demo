# 🎯 学习计划助手 - 演示项目

这是一个学习计划制定小程序的演示版本，展示了完整的UI界面和交互体验，但不含真实的后端逻辑和敏感数据。

## 🌟 项目特色

- ✅ **完整UI展示** - 展示所有核心页面的界面设计
- ✅ **交互体验** - 模拟真实的用户操作流程
- ✅ **多端适配** - 支持H5、微信小程序等多平台
- ✅ **安全演示** - 不含真实API和后端逻辑
- ✅ **易于部署** - 可直接部署到GitHub Pages

## 📱 功能演示

### 🏠 首页功能
- 学习计划卡片展示
- 快速开始学习按钮
- 学习进度概览
- 热门课程推荐

### 📅 计划管理
- 学习计划列表
- 进度跟踪显示
- 计划状态管理（进行中/已完成/待开始）
- 模拟创建新计划

### 👤 个人中心
- 用户头像和基本信息
- 学习统计数据（已完成/进行中/学习时长）
- 设置选项
- 关于页面

### 🔐 登录演示
- 手机号+验证码登录
- 微信登录演示
- QQ登录演示
- 快速体验模式（无需登录）

## 🚀 快速开始

### 环境要求
- Node.js >= 12.0.0
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发环境启动
```bash
# 启动H5开发服务器
npm run dev:h5

# 启动微信小程序开发
npm run dev:mp-weixin
```

### 生产环境构建
```bash
# 构建H5版本
npm run build:h5

# 构建微信小程序版本
npm run build:mp-weixin
```

## 🎨 技术栈

- **框架**: UniApp (Vue2)
- **样式**: CSS3 + Flexbox
- **图标**: Emoji + 系统图标
- **构建**: Webpack
- **部署**: 支持多平台发布
