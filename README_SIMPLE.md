# 学习计划助手演示项目

这是一个学习计划制定小程序的演示版本，展示了完整的UI界面和交互体验。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发环境启动
```bash
# 启动H5开发服务器
npm run dev:h5
```

### 生产环境构建
```bash
# 构建H5版本
npm run build:h5
```

## 📱 功能演示

- 🏠 **首页功能** - 学习计划卡片展示
- 📅 **计划管理** - 学习计划列表和进度跟踪
- 👤 **个人中心** - 用户信息和统计数据
- 🔐 **登录演示** - 多种登录方式演示

## ⚠️ 重要说明

- 🔒 **安全提示**: 本演示项目不含真实API接口和后端逻辑
- 🚫 **数据限制**: 所有数据均为模拟数据
- 🎮 **体验模式**: 提供快速体验模式，无需真实登录

## 🎯 使用场景

- 🏢 **项目展示** - 向客户展示产品原型
- 🎓 **教学演示** - 用于教学UniApp开发
- 📊 **UI参考** - 作为UI设计参考
- 🔍 **功能测试** - 测试界面和交互流程
